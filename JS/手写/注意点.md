## 节流，一段时间内只能运行一次

只需要**一个**定时器，时间计算的时候需要重置定时器

```javascript
// 第一个部分
if (timer) {
  clearTimeout(timer);
  timer = null;
}
```

```javascript
// 第二个部分定时器：这里需要将timer设置为null
if (tailing && timer === null) {
  timer = setTimeout(() => {
    run(context, args);
    timer = null;
  }, wait);
}
```

## 防抖

需要**两个**定时器

```javascript
// 第一个：
timer = setTimeout(() => {
  timer = null;
}, wait);
```

```javascript
// 第二个：这里不需要设置timer为null
timer = setTimeout(() => {
  run(context, args);
}, wait);
```
